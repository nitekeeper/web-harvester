1 · New component: EditableLabel

A small, reusable click-to-edit text widget. Lives at src/presentation/components/EditableLabel.tsx.

Static state (default)
📁 Obsidian Vault Primary ★ Remove

Hover label → dashed underline hint. Click → edit mode.

Edit mode (after click)
📁 ↵ save · Esc cancel

Blur also saves. Empty/whitespace → reverts to original.

Props
value: string  ·  onCommit: (next: string) => void  ·  className?: string
Enter / blur → commits if value changed and non-empty  ·  Esc → cancels, reverts  ·  Empty trim → reverts (no-op save)

3 · Data flow

User clicks label
  → EditableLabel enters edit mode (local state)
User types new name, presses Enter
  → onCommit(newLabel) called
  → NameStack passes up to DestinationRow
  → DestinationRow calls onRename(id, newLabel)
  → useDestinationHandlers.onRename() → storage.update(id, {label})
  → refresh() → useSettingsStore updated → list re-renders
